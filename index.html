ORIGINAL PROGRAMMING LANGUAGE

Slay turns code into a stage.

Slay is a fresh scripting language with a custom editor, `.sly` scripts, fast run panels, commandbook help, and a visual coding experience built for creators who want their tooling to feel alive.

Download for Windows Explore the language

Own Syntax

Slay uses its own keyword voice: `say`, `stash`, `spark`, `send`, `loop`, and more. It feels readable, quick, and expressive without cloning another language.

Desktop Studio

Multi-tab scripts, output panel, local save/open support, visual settings, and a command index are built directly into the app.

.sly Files

Create scripts, save them to your machine, reopen them later, and build your own Slay code library.

WHY SLAY

For coding that looks sharp and feels distinct.

The current release ships as a working first version of the language runtime and editor. It is designed to grow into a bigger ecosystem with richer packages, deeper syntax, and stronger compile/runtime features.
