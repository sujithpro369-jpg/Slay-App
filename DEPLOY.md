# Deploy Slay Website

The website is static and ready for Vercel.

## Steps

1. Upload the `Website` folder to a repo or import it directly into Vercel.
2. Set the project root to `Website`.
3. Deploy.
4. After generating a Windows build, place the installer at `Website/downloads/Slay-Setup.exe`.

## URL Note

`slaypl.local` is not a public domain. Use one of these approaches:

- local-only host mapping on your own machine if you want a private dev alias
- a real public domain such as `slaypl.com` if you want 24/7 public access
- a hosted subdomain such as `slaypl.vercel.app` for immediate deployment
